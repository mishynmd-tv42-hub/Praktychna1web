##  Опис проєкту

Односторінковий інформаційний веб-сайт сонячної ферми **«Сонячна Долина»** (35 МВт, Миколаївська обл.).

Сайт містить:
- Навігаційне меню (Bootstrap Navbar) з посиланнями на розділи
- Hero-секцію із заголовком, підзаголовком і статистикою
- Розділ «Про об'єкт» (2 абзаци + картки переваг)
- Таблицю технічних параметрів (12 рядків)
- Структурну схему та фото об'єкта
- Контактну інформацію
- Адаптивний дизайн для мобільних пристроїв
